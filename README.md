# Lottery Game project
This project is a scrip developing on Python to simulate the Mexican Lottery Game that involves 54 cards and allows create your own play card personalized.
The program have a dealer mode too that shows randomly cards in determinate periods of time that you can pick automatizanding your game.
## Technologies
A list of technologies used within the project:
- Pycharm 2025.
- Python 3 or superior. (Pandas Library, Tkinter and Customtkinter)
## Installation and use
git clone https://github.com/Samir-Garcia/P1-github.git
cd lotery_game

The excecutable its only that you need to make it works, it weighs approximately 300 MB, but we recommends get at least 1 GB of free storage.
For a better experiences its recommend to use a FHD screen.
